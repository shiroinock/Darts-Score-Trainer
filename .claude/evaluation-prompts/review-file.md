## 評価タスク

このサブエージェントの実行結果を評価し、`.claude/agents/review-file.md` または該当する観点ファイル（`.claude/review-points/*.md`）の指示内容に改善すべき点があれば、そのファイルに追記してください。

## 評価観点

### 1. レビュー観点の適切性
- 適用したレビュー観点が対象ファイルに適しているか
- レビュー観点のチェック項目を全て確認しているか
- 重要度の高い項目を見逃していないか

### 2. 指摘の具体性と建設性
- 指摘内容が具体的か（ファイル名、行番号を含むか）
- 改善方法が明示されているか
- 批判的でなく、建設的な表現か

### 3. 判定（PASS/WARN/FAIL）の妥当性
- 判定基準が一貫しているか
- 深刻度が適切に反映されているか
- 過度に厳しい/甘い判定になっていないか

### 4. 見逃しと過検出
- 実際の問題を見逃していないか
- 問題でないものを指摘していないか（false positive）

## 改善アクション

評価結果に基づき、以下のいずれかを実行してください：

### 問題がある場合

#### review-file.md の改善
一般的なレビュープロセスの問題：
- レビュー手順の明確化
- 判定基準の改善
- レポート形式の改善

#### 観点ファイル（.claude/review-points/*.md）の改善
特定の観点に関する問題：
- チェック項目の追加
- 判定基準の調整
- 具体例の追加

### 問題がない場合
- 何もしないでください

## 改善対象ファイルの判断

- **プロセス全般の問題** → `.claude/agents/review-file.md`
- **TypeScript観点の問題** → `.claude/review-points/typescript.md`
- **座標系観点の問題** → `.claude/review-points/coordinate-system.md`
- **仕様準拠観点の問題** → `.claude/review-points/specification.md`

## 重要な注意事項

- サブエージェント(Task tool)を再帰的に呼び出さないでください
- Edit ツールと Read ツールのみを使用してください
- 複数のファイルを改善する必要がある場合は、それぞれに追記してください
