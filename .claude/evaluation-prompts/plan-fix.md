## 評価タスク

このサブエージェントの実行結果を評価し、`.claude/agents/plan-fix.md` の指示内容に改善すべき点があれば、そのファイルに追記してください。

## 評価観点

### 1. 問題分類の正確性
- fixable（修正可能）と unfixable（修正不可能）の判定が適切か
- 問題の深刻度を正しく評価しているか
- 問題の根本原因を特定できているか

### 2. 修正計画の実行可能性
- 修正手順が具体的で実行可能か
- ファイル名、行番号などの詳細が含まれているか
- 修正の影響範囲を考慮しているか

### 3. 優先度付けの妥当性
- 問題の優先順位が適切か
- クリティカルな問題を最優先しているか
- 修正の依存関係を考慮しているか

### 4. 修正指示の明確性
- implement エージェントが理解できる形式か
- 修正の「何を」「どのように」が明確か
- テストケースへの影響が考慮されているか

## 改善アクション

評価結果に基づき、以下のいずれかを実行してください：

### 問題がある場合
`.claude/agents/plan-fix.md` に以下を追記：

1. **判定基準の明確化**
   - fixable/unfixable の具体的な判定基準
   - 問題タイプ別の対処方針

2. **修正計画テンプレートの改善**
   - より詳細なテンプレート
   - 必須項目の明示

3. **具体例の追加**
   - 良い修正計画の例
   - 悪い修正計画の例

### 問題がない場合
- 何もしないでください

## 重要な注意事項

- サブエージェント(Task tool)を再帰的に呼び出さないでください
- Edit ツールと Read ツールのみを使用してください
- ファイルの変更は `.claude/agents/plan-fix.md` のみに限定してください
