## 評価タスク

このサブエージェントの実行結果を評価し、`.claude/agents/test-check.md` の指示内容に改善すべき点があれば、そのファイルに追記してください。

## 評価観点

### 1. テストの実行
- `npm run test:run` が正しく実行されているか
- テスト結果が適切に取得されているか

### 2. 結果判定の正確性
- 成功・失敗の判定が正しいか
- テスト数のカウントが正確か
- 失敗したテストの詳細が含まれているか

### 3. 出力フォーマット
- status (PASSED/FAILED) が明確か
- testCount、passed、failed が含まれているか
- 失敗したテストの詳細が含まれているか

## 改善アクション

評価結果に基づき、以下のいずれかを実行してください：

### 問題がある場合
`.claude/agents/test-check.md` に以下を追記：

1. **実行方法の改善**
   - 例: 「テスト数を正確にカウントしてください」

2. **出力フォーマットの改善**
   - 例: 「失敗時は失敗したテストの詳細を全て含めてください」

### 問題がない場合
- 何もしないでください

## 重要な注意事項

- サブエージェント(Task tool)を再帰的に呼び出さないでください
- Edit ツールと Read ツールのみを使用してください
- ファイルの変更は `.claude/agents/test-check.md` のみに限定してください
