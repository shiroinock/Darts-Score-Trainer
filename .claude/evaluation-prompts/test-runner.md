## 評価タスク

このサブエージェントの実行結果を評価し、`.claude/agents/test-runner.md` の指示内容に改善すべき点があれば、そのファイルに追記してください。

## 評価観点

### 1. テスト実行が正確か
- `npm test` コマンドが正しく実行されているか
- テスト結果が正しく取得できているか
- エラーメッセージが適切にパースされているか

### 2. 状態判定（Red/Green）の正確性
- テスト結果から Red/Green を正しく判定しているか
- 期待する状態（RED_EXPECTED/GREEN_EXPECTED）との比較が正しいか
- 判定結果（SUCCESS/FAILURE）が適切か

### 3. エラーメッセージの有用性
- 失敗したテストの詳細が明確に報告されているか
- エラー診断が具体的で actionable か
- 次ステップの提案が適切か

### 4. レポート品質
- サマリー情報（成功数、失敗数、実行時間）が正確か
- 失敗の原因分析が含まれているか
- 次に何をすべきかが明確か

## 改善アクション

評価結果に基づき、以下のいずれかを実行してください：

### 問題がある場合
`.claude/agents/test-runner.md` に以下を追記：

1. **不足していたガイドライン**
   - 例: 「テスト結果のパース方法を改善してください」
   - 例: 「エラー診断にファイル名と行番号を含めてください」

2. **エラーパターンの追加**
   - 新しいエラーパターンとその診断方法

3. **次ステップ提案の改善**
   - より具体的な提案例

### 問題がない場合
- 何もしないでください

## 重要な注意事項

- サブエージェント(Task tool)を再帰的に呼び出さないでください
- Edit ツールと Read ツールのみを使用してください
- ファイルの変更は `.claude/agents/test-runner.md` のみに限定してください
