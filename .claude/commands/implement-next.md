# 次のTODOタスクを実装

TODO.mdから未完了のタスクを1つ選び、implementサブエージェントに実装を依頼してください。

## 手順

1. **TODO.mdを読み込み**、最初の未完了タスク（`- [ ]`）を特定する
2. **COMPLETE_SPECIFICATION.md**から該当タスクの詳細仕様を抽出する
3. **implementサブエージェント**（subagent_type: implement）をTaskツールで起動し、以下を含むプロンプトを渡す：
   - 実装すべきタスクの明確な説明
   - COMPLETE_SPECIFICATION.mdからの関連仕様
   - 期待される成果物（ファイルパス、関数名など）
   - 依存関係（既存のファイルやモジュールがあれば）
4. サブエージェント完了後、**SubagentStop hookが自動実行**され、evaluate-subagent.shがimplement.mdを評価・改善する
5. **TODO.mdを更新**：完了したタスクに`[x]`をマークし、進捗表を更新する
6. **結果を報告**：実装されたファイル、hookによる改善内容（あれば）をユーザーに報告

## 注意事項

- 依存関係のあるタスクは、先行タスクが完了していることを確認してから実行
- エラーが発生した場合は、原因を特定してユーザーに報告
